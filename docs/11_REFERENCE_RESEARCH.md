## Claude research task

Before final visual polish, browse the pages above and update `docs/05_DESIGN.md` with:

- recurring color combinations;
- border and frame proportions;
- numeric display styles;
- bulb placement and chase direction;
- scenic transitions;
- product pedestal composition;
- visual elements that can be abstracted into original CSS/SVG;
- elements that should not be copied.

Do not download external images automatically. The uploaded references and original project SVGs are enough to build the first production design.
