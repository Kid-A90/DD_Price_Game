# Approved product shots only

Only exact, reviewed product cutouts belong here.

Each file must:
- show the exact purchased product;
- remove all store names, receipts, price stickers, hands, and room background;
- preserve the product's actual color, proportions, labels, packaging, and artwork;
- be approved side-by-side against the source photo;
- use a neutral white or transparent background.

Never copy anything from `private/source-products` directly into this folder.
